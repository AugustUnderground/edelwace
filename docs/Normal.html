data Normal Source #

Creates a normal (also called Gaussian) distribution parameterized by loc and scale. Example:

>>> m = Normal (asTensor ([0.0] :: [Float])) (asTensor ([1.0] :: [Float]))
>>> sample m []
Tensor Float [1] [-0.1205   ]
